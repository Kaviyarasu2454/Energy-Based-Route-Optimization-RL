# Energy-Based Route Optimization using Reinforcement Learning

This project focuses on optimizing routes for electric vehicles based on energy consumption using Reinforcement Learning (Q-Learning).

## Features
- Real-world road network using SUMO
- Energy-based routing instead of shortest path
- Comparison with Dijkstra and A*
- Congestion modeling
- Battery tracking

## Technologies Used
- Python
- SUMO
- Reinforcement Learning (Q-Learning)
- Matplotlib

## How to Run
1. Install dependencies
2. Run `main.py`
3. Select start and destination on map

## Results
- RL can achieve energy-efficient routes in certain scenarios
- Compared with Dijkstra and A*

## Future Scope
- Dynamic traffic
- Better reward tuning
- Real-time data integration
